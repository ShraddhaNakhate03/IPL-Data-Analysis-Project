# 🏏 IPL Data Analysis Project

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA) on Indian Premier League (IPL) data** to uncover insights about teams, players, and match outcomes. Using Python and data visualization techniques, the project analyzes historical IPL matches to identify performance trends, winning patterns, and key statistics.

The goal of this project is to demonstrate **data analysis, visualization, and storytelling skills using real-world sports data**.

---

## 🎯 Objectives

* Analyze **team performance across IPL seasons**
* Identify **top performing batsmen and bowlers**
* Study **match-winning patterns**
* Explore **venue impact on match outcomes**
* Visualize important statistics using charts and graphs

---

## 📂 Dataset

The dataset contains historical IPL match data including:

* Match results
* Teams
* Players
* Venues
* Toss decisions
* Match statistics

Typical datasets used:

* `matches.csv`
* `deliveries.csv`

---

## 🔎 Exploratory Data Analysis

The analysis includes:

### Team Analysis

* Total matches played by each team
* Win percentage by team
* Toss impact on match results

### Player Performance

* Top run scorers
* Best bowlers
* Most sixes and fours

### Venue Analysis

* Matches played at different stadiums
* Venue influence on match outcomes
